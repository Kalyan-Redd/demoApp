# 🏠 Mortgage Repayments Calculator

An interactive Streamlit app that calculates monthly mortgage repayments, total interest paid, and visualizes the full loan repayment schedule over time.

---

## 📌 Problem Statement

Understanding the true cost of a home loan — monthly payments, total interest, and how the balance decreases over time — is often confusing without a proper amortization breakdown. This tool gives users instant, transparent insight into their mortgage before committing to a loan.

---

## ✅ What I Built & Achieved

- Built a **mortgage repayment calculator** using the standard amortization formula to compute fixed monthly payments
- Calculated **total repayment amount** and **total interest paid** over the full loan term
- Generated a **month-by-month amortization schedule**, tracking principal, interest, and remaining balance for every payment
- Visualized the **declining loan balance over time** using an interactive line chart, grouped by year
- Designed a clean **two-column input layout** for home value, deposit, interest rate, and loan term
